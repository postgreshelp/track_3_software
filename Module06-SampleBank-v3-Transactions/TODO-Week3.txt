[ ] 3. Write CREATE TABLE transactions in transactions-schema.sql:

       - to_user_id INT NOT NULL REFERENCES users(user_id)
       - amount DECIMAL(12,2) NOT NULL CHECK (amount > 0)
       - created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP

===============================================================
TEST YOUR IMPLEMENTATION
===============================================================

1. Create two test users:
   curl -X POST http://localhost:8080/register \
     -H "Content-Type: application/json" \
     -d '{"username":"alice","email":"alice@test.com","password":"pass123"}'

   curl -X POST http://localhost:8080/register \
     -H "Content-Type: application/json" \
     -d '{"username":"bob","email":"bob@test.com","password":"pass456"}'

2. Check balances:
   curl http://localhost:8080/balance/alice
   curl http://localhost:8080/balance/bob

3. Transfer money:
   curl -X POST http://localhost:8080/transfer \
     -H "Content-Type: application/json" \
     -d '{"fromUsername":"alice","toUsername":"bob","amount":"250.00"}'

4. Verify balances changed:
   curl http://localhost:8080/balance/alice  (should be $750.00)
   curl http://localhost:8080/balance/bob    (should be $1250.00)

5. Verify transaction recorded:
   psql -U postgres -d samplebank -c "SELECT * FROM transactions;"
